# Wyoming DAO LLC Formation Checklist

## Pre-Filing
- [ ] Choose company name: `[Name] DAO LLC`
- [ ] Verify name availability at wyobiz.wyo.gov
- [ ] Select registered agent service
- [ ] Contract with registered agent (get WY address)
- [ ] Prepare management article text
- [ ] Create MetaMask wallet for AI corp

## Filing at wyobiz.wyo.gov
- [ ] Business Entity Type: LLC (Domestic)
- [ ] Additional Designation: DAO
- [ ] Name: `[Name] DAO LLC` (confirmed twice)
- [ ] Delayed Effective Date: blank (immediate)
- [ ] Registered Agent: searched and selected
- [ ] Principal Address: registered agent's WY address
- [ ] Mailing Address: same as above
- [ ] Organizer: human name + address
- [ ] Identifier: No (30-day amendment)
- [ ] Notice of Restrictions: No (auto-included)
- [ ] Management Article: algorithmically managed text
- [ ] DAO election article: confirmed
- [ ] Signature: human organizer
- [ ] Payment: $100

## Post-Filing (Within 30 Days)
- [ ] Receive confirmation from Secretary of State
- [ ] Deploy smart contract (Gnosis Safe)
- [ ] File amendment with contract address
- [ ] Apply for EIN (Form SS-4)

## Ongoing Compliance
- [ ] Annual Report: due anniversary month ($60/year)
- [ ] FinCEN BOI report: file beneficial ownership
- [ ] Federal tax return: pass-through to organizer
- [ ] Registered agent renewal: annual
