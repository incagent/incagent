example.py
from incagent import IncAgent, Contract, ContractTerms # Create two agent-run companies vendor = IncAgent(name="IncAgent Vendor", role="seller") buyer = IncAgent(name="Northstar Buyer", role="buyer") # Define the offer contract = Contract( title="Outbound Campaign System", terms=ContractTerms( price=5_000, currency="USD", duration_days=30, ), ) # Negotiate, sign, pay — no humans required result = await vendor.transact( counterparty=buyer, contract=contract, ) print(result.status) # "completed" print(result.revenue) # 5000.0 print(result.tax_filed) # True
How it works
01 — Build
Create the offer
An internal agent builds the service package, sets terms, and activates it for counterparty discovery.
02 — Transact
Accept the terms
A counterparty agent reviews the proposal and signs the contract. No human approval on either side.
03 — Get paid
Revenue recorded
Invoice is issued, payment is settled, and revenue is logged. Tax records are generated automatically.
04 — Operate
Company continues
Growth, finance, and compliance agents continue the workflow. The company runs without operators.
What this is
IncAgent is not an assistant.
It is a protocol layer for agent-run companies.

It helps internal agents and counterparty agents create, approve, execute, and record business activity — from the first offer to the final tax record.

Corporate identity per agent
Each agent holds a verifiable identity — name, ID, cryptographic key — enabling trusted counterparty relationships.
Autonomous negotiation
Agents propose, counter, and agree without human input. Rounds are bounded, logged, and auditable.
End-to-end transaction record
Every contract, invoice, payment, and tax filing is written to an immutable local ledger.
Why now

Most AI products help a person do work. The next layer is when agents do the work of the company itself.

The tools are already here — agents that write code, run campaigns, manage finances. What's missing is a protocol for them to transact with each other. IncAgent is that protocol. Open source, MIT licensed.
