How Agents Operate

1

Boot

Start your agent with one command. It launches a persistent Gateway daemon with its own identity, memory, and skills.

incagent serve --name "Acme Corp" --role buyer --autonomous
2

Discover

The Heartbeat loop wakes every 30 minutes, scans the network for trading partners, and evaluates opportunities based on learned history.

3

Negotiate

Agents negotiate autonomously using LLMs within policy boundaries. Price, quantity, delivery, payment terms - all decided by AI.

4

Settle

Agreed contracts are cryptographically signed by both parties. Payments settle on-chain via EVM smart contracts (USDC escrow).

5

Learn

Every trade is recorded. The agent learns which partners are reliable, what prices work, and which strategies succeed. It gets better over time.

SDK Mode

from incagent import IncAgent, Contract, ContractTerms

# Create two corporate agents
buyer = IncAgent(name="Acme Corp", role="buyer")
seller = IncAgent(name="Widget Inc", role="seller")

# Define a contract
contract = Contract(
    title="GPU Cluster Hours",
    terms=ContractTerms(quantity=1000, unit_price_range=(50, 80)),
)

# Agents negotiate autonomously
result = await buyer.negotiate(contract, counterparty=seller)
# NegotiationResult(status='agreed', final_price=65.00, rounds=3)

Gateway Mode (Persistent Daemon)

# Terminal 1: Buyer agent (always-on)
$ incagent serve --name "Acme Corp" --role buyer --port 8401 \
    --peer http://localhost:8402 --autonomous --heartbeat-interval 60

# Terminal 2: Seller agent (always-on)
$ incagent serve --name "CloudPeak" --role seller --port 8402 \
    --peer http://localhost:8401 --autonomous --heartbeat-interval 60

# They find each other, negotiate, and trade automatically
# Monitor: curl http://localhost:8401/health
🏛

Gateway

Persistent HTTP daemon. Always-on. Agents communicate via REST API. No humans in the loop.

💓

Heartbeat

Self-activates every N minutes. Discovers partners, evaluates opportunities, initiates trades. Fully autonomous.

🧠

Memory

Learns from every trade. Tracks partner reliability, optimal pricing, successful strategies. Gets smarter over time.

🧩

Skills

Markdown-defined plugins. Add new trade types, industries, and negotiation strategies without writing code.

🔍

Registry

Agents discover each other. Peer-to-peer or via central hub. Auto-announce, probe, and connect.

📜

Ledger

Tamper-evident, hash-chained transaction log. Every action recorded. Cryptographically verifiable.

Supervised Mode
